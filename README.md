# Sapient — Distraction‑Free Video Learning System (iOS)

Sapient is an iOS-native **Bring Your Own Content** video learning system. It transforms isolated video files into a structured, distraction-free curriculum with built-in progress tracking, AI-assisted summaries, and retention mechanics.

## Why Sapient?

Mainstream video platforms (e.g., YouTube) are optimized for engagement loops and distraction. Standard media players, on the other hand, lack learning context: course structure, progress tracking, and retention support.

Sapient aims to provide a focused learning environment for your *own* downloaded or recorded content.

## Target audience

- **Self-taught learners** upskilling via downloaded courses, conference talks, and deep-dive technical videos—especially those building a personal knowledge library and needing offline access.
- **Students** managing heavy loads of recorded lectures.

## Core features (MVP)

### 1) Content ingestion
- **Local import:** Import `.mp4`, `.mov`, and other common formats via the iOS Files app.
- **Cloud drive import:** Import from Google Drive, iCloud Drive, and Dropbox.

### 2) Contextual organization & curriculum mapping
- **Course grouping:** Create “Courses” / “Topics” and arrange videos into a sequential syllabus.
- **Dynamic playlists:** Group videos by tags (e.g., “Must Watch”, “Review Later”).

### 3) Advanced playback & learning environment
- **Distraction-free player:** Minimal, gesture-driven custom `AVPlayer` UI.
- **Picture-in-Picture (PiP):** Background listening and multitasking while learning.
- **Offline-first:** Imported/downloaded videos remain usable without an internet connection.

### 4) Progress tracking & timeline
- **Granular progress:** Completion progress per-video and per-course.
- **Resume playback:** Remembers the exact timestamp for each video.
- **Activity timeline:** “Recently Played” and “Continue Learning” views.

### 5) Retention & engagement
- **Smart reminders:** Local notifications prompting you to continue learning after inactivity.
- **On-device AI summarization:** Local transcription + concise summaries and key bullets without server calls.

## Roadmap (Post‑MVP)

- **Phase 2:** Cloud sync across iOS / iPadOS / macOS devices.
- **Phase 3:** Peer-to-peer sharing of course structure (metadata, notes, links).
- **Phase 4:** Native Markdown notes attached to video timestamps.
